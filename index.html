▌ ABOUT LAWBWORLD

Lawbworld is a 3D pixel-art city built from live repositories on the gitlawb decentralized git network. Every repo becomes a building — its size, height, and rarity derived from on-chain file weight.

How it works:

  • The city syncs from gitlawb.com/node on load — every repo placed into a procedurally generated voxel skyline.
  • Building height · log of repo file size in bytes
  • Building width · derived from commit count + branch count
  • Window glow pattern · derived from the DID public key hash
  • Rarity tier · 10 Legendary, 50 Epic, 100 Rare, rest Common / Uncommon
  • Transmitter towers at city edges represent p2p peers

Architecture: Frontend (this HTML) · Backend (Express REST API at /api, full live gitlawb sync) · UUPS-upgradeable LawbBuilding ERC-721 on Base L2. Mint = 0.00028 ETH. Upgrade = burn $LAWBWORLD, deployed via the Bankr launchpad.
